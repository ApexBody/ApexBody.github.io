Laptop Specifications

Minimum Laptop Configuration


Processor Recent generation (>=8) Intel® Core™ i5 or AMD Ryzen 3
Memory and HDD - DDR4 Model: Minimum 8GB RAM for Mac OR 8GB DDR4 2666MHz (For Windows, Preferable 16GB and above)
- SSD (M.2/3 PCIe NVMe) instead of SATA, 256GB (For Mac) OR 512GB (For Windows for Dual Boot)
- Built in on-board graphics unit
Operating System Ubuntu 22.04 (Will be dual Booted during Induction Session for Windows Machines) OR MacOS
Mac systems may not be supported for some courses in the institute.
Ports and Connectivity Ethernet port, HDMI, Mic In, RJ45, USB Ports (A and C), Wifi
If laptop doesnt have an Ethernet port, etc. a dock/adapter (USB to Ethernet) can be used
Others Try carrying a USB pendrive (>=16GB) to flash it for installing Ubuntu


P.S. (few notes from the Apex):
Above are the official college laptop requirements, please make your own informed decisions on what system you choose to use using benchmarks and information online.
Also, in case you do not have an Ubuntu system, there will be a session during induction to help you install and boot your laptop to Ubuntu.
